# CurrencyTravelBot
Travel Wallet Bot — Telegram-бот для управления финансами в поездках. Позволяет создавать несколько кошельков с разными валютами, отслеживать расходы и автоматически конвертировать суммы через exchangerate.host. Поддерживает ручной ввод курса, историю операций и удобную навигацию через inline-кнопки. Данные хранятся в SQLite.
