# CurrencyTravelBot

Telegram-бот «Миникошелёк для путешественника» — управление финансами в поездках с конвертацией через **api.exchangerate.host**. Данные хранятся локально в SQLite.

## Возможности

- Создание нескольких путешествий (валютная пара + курс + баланс).
- Проверка доступных валют и текущего курса через API, ручной ввод курса (например, по обменнику).
- Начальный баланс: ввод суммы в домашней валюте, конвертация через API в валюту поездки.
- Любое сообщение с числом — сумма расхода в валюте пребывания; конвертация в домашнюю и кнопки «Учесть как расход? Да / Нет».
- Inline-меню: Создать путешествие, Мои путешествия, Баланс, История расходов, Изменить курс.
- Слэш-команды: `/newtrip`, `/switch`, `/balance`, `/history`, `/setrate`.

## Установка и запуск

1. Клонировать репозиторий и перейти в каталог.
2. Создать виртуальное окружение и установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Настроить конфигурацию. Скопировать `.env.example` в `.env` и указать:
   - `EXCHANGERATE_ACCESS_KEY` — ключ API с [exchangerate.host](https://exchangerate.host/)
   - `BOT_TOKEN` — токен бота от [@BotFather](https://t.me/BotFather)

4. Запуск:

   ```bash
   python bot.py
   ```

База SQLite создаётся автоматически в файле `bot_data.db` в каталоге проекта.
